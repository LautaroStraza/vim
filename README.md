# vimrc
Vimrc configuraciones para vim.

Usa los siguientes plugins:
```
  Plugin 'ervandew/supertab'
  Plugin 'scrooloose/nerdtree'
  Plugin 'ap/vim-buftabline'
  Plugin 'vim-syntastic/syntastic'
```
